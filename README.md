# ACEest Fitness & Gym - Automated CI/CD Pipeline

## Project Overview
This repository contains the foundational Flask web application and the automated CI/CD pipeline infrastructure for ACEest Fitness & Gym. It demonstrates professional DevOps methodologies including Version Control, Containerization (Docker), and pipeline orchestration.

## Local Setup & Execution
To run this application on your local machine, follow these steps:

1. **Clone the repository:**
  ```bash
   git clone https://github.com/2024ht66503-shash/devops-assignment-Shashwat-Sharma
   cd devops-assignment-Shashwat-Sharma

2. **Install dependencies::**
  ```bash
   pip install -r requirements.txt

3. **Run the Flask application::**
  ```bash
   python app.py

The app will be accessible at http://localhost:5000 and http://localhost:5000/services
